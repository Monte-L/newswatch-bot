# NewsWatch Bot

NewsWatch Bot is a Linux-based Python project designed to collect, store, and organize news from RSS feeds.

The project starts as a command-line RSS collector and will gradually evolve into a local news aggregation platform with a web dashboard built using FastAPI and HTML.

## Project Goal

The goal of this project is to build an automated news collection system using:

- Python
- RSS feeds
- SQLite
- Logs
- Linux automation
- FastAPI
- HTML dashboard

The final objective is to centralize interesting news from multiple sources into a single local database and provide a simple dashboard to browse, filter, and refresh the collected news.

## Current Features

- Load RSS feed URLs from a text file
- Parse RSS feeds using Python
- Extract article title, link, source, and published date
- Store articles in a local SQLite database
- Prevent duplicate articles using a SHA-256 hash based on the article link
- Track when each article was collected
- Print only new articles in the terminal

## Article Metadata

The bot stores the following fields:

| Column | Description |
|---|---|
| id | Unique article identifier generated from the article link |
| title | Article title |
| link | Original article URL |
| source | RSS feed source |
| category | Article category generated by keyword matching |
| summary | Short summary extracted from RSS |
| image_url | Image URL extracted from RSS when available |
| published | Publication date from the RSS feed |
| fetched_at | UTC timestamp when the bot collected the article |
